=================================
Electronic Office File Management
=================================

Module to link the input and output record (crm_lettermgmt) and electronic files
(crm_filemgmt) with the electronic office electronic files and communications.

Description
===========

It allows to link an input or output record with a communication (input) or a
notification (output)  and electronic file with a file.
